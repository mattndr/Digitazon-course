<body>

This tag contains the content of the page that is displayed on screen.

<head>

This tag contains the following informations:

  • meta-informations,
  • page title,
  • (opt) links to css files,
  • (opt) page styles via a set of css rules defined within a <style> tag;

<html>

Root element of an HTML document.
